 #AI Worker Burnout and Attrition Risk Analysis#

Project Overview

This project analyzes the AI Worker Burnout and Attrition Risk dataset to understand the factors that contribute to employee burnout and potential attrition among AI professionals. The goal of this project is to explore patterns in workplace conditions, workload, and psychological factors using data analysis and machine learning techniques.

The project focuses on Exploratory Data Analysis (EDA) and Clustering techniques to identify meaningful patterns and group employees based on burnout risk.

Dataset

Dataset Source:
https://www.kaggle.com/datasets/nudratabbas/ai-worker-burnout-and-attrition-risk-dataset
The dataset contains survey responses from AI professionals, including information about workload, stress levels, job satisfaction, work environment, and mental health indicators. These factors help analyze employee burnout and attrition risk.

Project Objectives

The main objectives of this project are:
Perform Exploratory Data Analysis (EDA) to understand the dataset
Identify key features contributing to burnout
Visualize relationships between workplace factors and burnout levels
Apply clustering techniques to group employees with similar burnout patterns
Discover insights that could help organizations improve employee well-being

Exploratory Data Analysis (EDA)
EDA will be used to better understand the dataset and identify important patterns. The following steps will be performed:
Data cleaning and preprocessing
Handling missing values
Statistical summaries of the dataset
Feature distribution analysis
Correlation analysis between variables
Data visualization using charts and graphs
Visualization techniques may include:
Histograms
Box plots
Heatmaps
Correlation matrices
Pair plots

Clustering Approach

Clustering will be used to group employees with similar burnout characteristics. This helps identify different burnout risk groups.
The clustering methods that may be used include:
K-Means Clustering
Hierarchical Clustering
These methods will help identify patterns such as:
High burnout risk employees
Moderate burnout groups
Low burnout risk employees
